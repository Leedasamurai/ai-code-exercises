## **Selected Algorithm**

**Task Priority Scoring Algorithm (calculateTaskScore)**

This function calculates a weighted importance score for tasks based on priority, due date, status, tags, and recency.

---

# **Original Code**

```jsx
js
function calculateTaskScore(task) {
  const priorityWeights = {
    [TaskPriority.LOW]: 1,
    [TaskPriority.MEDIUM]: 2,
    [TaskPriority.HIGH]: 3,
    [TaskPriority.URGENT]: 4
  };

  let score = (priorityWeights[task.priority] || 0) * 10;

  if (task.dueDate) {
    const now = new Date();
    const dueDate = new Date(task.dueDate);
    const daysUntilDue = Math.ceil((dueDate - now) / (1000 * 60 * 60 * 24));

    if (daysUntilDue < 0) {
      score += 30;
    } else if (daysUntilDue === 0) {
      score += 20;
    } else if (daysUntilDue <= 2) {
      score += 15;
    } else if (daysUntilDue <= 7) {
      score += 10;
    }
  }

  if (task.status === TaskStatus.DONE) {
    score -= 50;
  } else if (task.status === TaskStatus.REVIEW) {
    score -= 15;
  }

  if (task.tags.some(tag => ["blocker", "critical", "urgent"].includes(tag))) {
    score += 8;
  }

  const now = new Date();
  const updatedAt = new Date(task.updatedAt);
  const daysSinceUpdate = Math.floor((now - updatedAt) / (1000 * 60 * 60 * 24));

  if (daysSinceUpdate < 1) {
    score += 5;
  }

  return score;
}
```

# **Insights & Improvements Identified Using Prompt 2**

After running Prompt 2 (Intent & Logic Explanation), I identified the following deeper insights:

---

## **High-Level Intent**

The function acts as a **deterministic ranking engine** that transforms multiple task attributes into a single numerical score.

It encodes business prioritization rules into additive weights.

---

## **Step-by-Step Logic**

1. Assign base score based on priority.
2. Increase score based on urgency (due date proximity).
3. Decrease score for completed or review tasks.
4. Increase score for high-impact tags.
5. Increase score for recently updated tasks.
6. Return final score.

---

## **Assumptions in the Code**

- task.tags always exists and is an array.
- task.updatedAt is a valid ISO date string.
- task.priority exists in the weight map.
- System timezone does not significantly affect date calculations.

---

## **Potential Edge Cases**

- task.tags undefined → runtime error.
- Invalid updatedAt → NaN in calculations.
- Timezone differences may affect overdue classification.
- Multiple new Date() calls inside function (minor inefficiency).
- Recalculation of score during sorting may impact performance.

---

## **Suggested Improvements (Without Changing Behavior)**

- Validate task.tags before calling .some().
- Validate updatedAt before date math.
- Inject now as parameter for testability.
- Extract scoring weights into configuration object.
- Normalize tag comparison to lowercase.

# Original Endpoint Code

```jsx
productRouter.get('/', async (req, res) => {
  try {
    const {
      category,
      minPrice,
      maxPrice,
      sort = 'createdAt',
      order = 'desc',
      page = 1,
      limit = 20,
      inStock
    } = req.query;

    const filter = {};

    if (category) {
      filter.category = category;
    }

    if (minPrice !== undefined || maxPrice !== undefined) {
      filter.price = {};
      if (minPrice !== undefined) filter.price.$gte = parseFloat(minPrice);
      if (maxPrice !== undefined) filter.price.$lte = parseFloat(maxPrice);
    }

    if (inStock === 'true') {
      filter.stockQuantity = { $gt: 0 };
    }

    const skip = (parseInt(page) - 1) * parseInt(limit);

    const sortOptions = {};
    sortOptions[sort] = order === 'asc' ? 1 : -1;

    const products = await ProductModel.find(filter)
      .sort(sortOptions)
      .skip(skip)
      .limit(parseInt(limit));

    const totalProducts = await ProductModel.countDocuments(filter);

    return res.status(200).json({
      products,
      pagination: {
        total: totalProducts,
        page: parseInt(page),
        limit: parseInt(limit),
        pages: Math.ceil(totalProducts / parseInt(limit))
      }
    });
  } catch (error) {
    return res.status(500).json({
      error: 'Server error',
      message: 'Failed to fetch products'
    });
  }
});
```
